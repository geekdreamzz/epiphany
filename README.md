# Epiphany
Open Source Ruby on Rails engine to build Custom Voice Assistants

## Usage
*Currently in Alpha 

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'epiphany', :git => "git://github.com:geekdreamzz/epiphany.git", :branch => "alpha"
```

And then execute:
```bash
$ bundle
```
